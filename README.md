# 蓮Type

蓮Typeはブラウザ用タイピング練習アプリです。

## 使い方

1. `index.html` をブラウザで開きます。
2. `開始` ボタン、またはスペースキーで練習を始めます。
3. `基礎`、`単語`、`英単語`、`短文`、`数字` のタブで練習内容を切り替えます。

`し` は `shi` / `si`、`じ` は `ji` / `zi` のように、よく使う別ローマ字入力にも対応しています。
`単語` には、契約・税金・年金・保険など、社会人の一般常識として使う漢字語も入っています。漢字を見て、読みをローマ字で打ちます。
`短文` は、ひらがなと漢字が混ざった文を表示します。表示文の読みをスペースなしのローマ字で打ちます。

`英単語` は60秒の連続練習です。画面に並んだ英単語を、左から順にそのまま打っていきます。
英単語は `english-dictionary.js` の辞書から出題されます。

ローカルサーバーで開く場合:

```powershell
cd "C:\Users\banxi\OneDrive\ドキュメント\c\ren-type-web"
node serve.js
```

そのあと `http://127.0.0.1:5178/` を開きます。
